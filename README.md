# Customer Order Analytics Pipeline

## Overview
Scalable ETL pipeline using PySpark, Spark, Hadoop, SQL, and Airflow.

## Features
- Batch ETL Processing
- Spark Transformations
- Workflow Automation
- Data Validation
- Analytics Reporting

## Tech Stack
- Python
- PySpark
- Apache Spark
- Hadoop
- Airflow
- SQL

python spark_jobs/transform.py
